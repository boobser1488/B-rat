## 📋 Описание

Проект представляет собой мощного агента для удалённого администрирования Windows-систем, управляемого через Telegram-бота. Написан на **Rust**, что обеспечивает высокую производительность, безопасность памяти и скрытность. Агент маскируется под легитимный системный процесс (`svchost.exe`) и обладает широчайшим спектром возможностей: от мониторинга системы и сбора данных до полного управления файловой системой, реестром, процессами и периферией.

Ключевая особенность — наличие **Watchdog (реаниматора)**, который автоматически перезапускает агента в случае его аварийного завершения, обеспечивая высокую устойчивость. Агент также содержит механизмы анти-отладки и обфускации для затруднения анализа.

## ✨ Возможности

### 🛠️ Основные
- Авторизация по ключу (по умолчанию `B-ratInRust??`)
- Поддержка команд через Telegram с разбором аргументов (в том числе в кавычках)
- Полное логирование действий (для отладки) в `%TEMP%\cmd_debug.log`
- Анти-отладка (обнаружение отладчика, песочницы, скрытие консоли)
- Автозагрузка через реестр Windows
- Self‑destruct (самоуничтожение с удалением файла после перезагрузки)

### 🔄 Watchdog (Реаниматор)
- Автоматический запуск при старте агента
- Мониторинг основного процесса (по PID)
- Перезапуск при краше
- Логирование событий в `%LOCALAPPDATA%\Microsoft\watchdog.log`
- Команды управления: `/wd start`, `/wd stop`, `/wd status`, `/wd logs`

### 📁 Файловая система
- Скачивание/загрузка файлов (с поддержкой больших файлов до 50 МБ)
- Создание, удаление, перемещение, переименование
- Рекурсивный обход папок, поиск по маске
- Скрытие/отображение файлов (атрибут `HIDDEN`)
- Подсчёт контрольных сумм (MD5, SHA1, SHA256, SHA512)
- Упаковка/распаковка ZIP-архивов

### 🖥️ Система и процессы
- Получение информации о системе (CPU, RAM, диски, сеть, аптайм)
- Мониторинг загруженности процессов в реальном времени
- Запуск произвольных программ и PowerShell-команд
- Управление службами и задачами планировщика
- Чтение/запись/удаление ключей реестра

### 📸 Захват и мультимедиа
- Скриншоты (весь экран или область)
- Фото с веб-камеры
- Запись с микрофона (WAV)
- Кейлоггер (лог нажатий клавиш)
- Мониторинг активного окна

### 📋 Буфер обмена
- Получение текущего содержимого
- Установка текста в буфер обмена
- Мониторинг изменений буфера обмена

### 🔑 Кража данных (мб не работает)
- Сбор cookie-файлов браузеров (Chrome, Firefox)
- Поиск файлов с паролями (Login Data Chrome)
- Извлечение сохранённых паролей Wi-Fi
- История браузера и закладки (Chrome)

### 🖱️ Управление периферией
- Эмуляция нажатий клавиш
- Печать текста
- Перемещение мыши, клики (левая/правая/средняя)
- Получение позиции курсора

### ⚡ Управление питанием
- Выключение, перезагрузка, гибернация, сон
- Блокировка рабочей станции, выход из системы
- Отмена запланированного выключения

### 🎉 Развлечения и прочее
- Открытие RickRoll в браузере
- Статистика использования команд
- Просмотр и очистка логов агента
- Информация о версии
